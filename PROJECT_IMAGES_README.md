## Spesifikasi Gambar
- **Format**: JPG atau PNG
- **Ukuran**: Minimal 800x600px (16:9 ratio recommended)
- **Kualitas**: High quality untuk tampilan yang baik
- **Nama file**: Harus sesuai dengan format di atas

## Fitur Slider
✅ **Navigation Buttons**: Tombol prev/next untuk navigasi
✅ **Dot Navigation**: Indikator slide aktif
✅ **Slide Counter**: Menampilkan posisi slide (1/7)
✅ **Touch/Swipe**: Support untuk mobile devices
✅ **Keyboard Navigation**: Arrow keys untuk navigasi
✅ **Auto-play**: Otomatis berganti slide setiap 5 detik
✅ **Hover Pause**: Auto-play berhenti saat hover
✅ **Responsive**: Menyesuaikan ukuran layar

## Cara Kerja
1. Setiap project memiliki slider dengan 7 gambar
2. Gambar pertama ditampilkan sebagai default
3. User dapat navigasi dengan tombol, dots, atau swipe
4. Counter menunjukkan posisi slide saat ini
5. Auto-play berjalan di background
6. Responsive untuk semua ukuran layar
